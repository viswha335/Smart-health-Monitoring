📋 How to Use Notifications

These notifications are integrated throughout the SHMA application. They automatically appear when:

✓ You save your profile information
✓ You update permission settings
✓ You log in or create an account
✓ Health alerts are triggered
✓ System updates occur

Auto-dismiss: Notifications automatically disappear after 3 seconds.

Manual close: Click the × button to dismiss immediately.

// Show a notification from any page:
showNotification('success', 'Title', 'Message');

// Types: 'success', 'error', 'info', 'warning'
