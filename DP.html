User Experience

Pairing Workflow Examples

Step-by-step pairing procedures for common device types on the VIS platform.

⌚

Pairing a Smartwatch

Duration: 2-3 minutes

  • Enable Bluetooth on both devices
  • Open VIS app → Add Device → Wearables
  • Select device from discovery list
  • Confirm PIN on watch display
  • Grant health data permissions
  • Complete: Real-time data syncing active
📱

Pairing a Smartphone

Duration: 1-2 minutes

  • Download VIS mobile app from store
  • Create/login to VIS account
  • Grant device permissions (location, health)
  • App automatically registers as primary device
  • Connect secondary devices via main app
  • Complete: Phone serves as hub
🏠

Pairing Smart Home Devices

Duration: 3-5 minutes

  • Ensure device is powered and nearby
  • VIS App → Smart Home → Add Device
  • Select device type from catalog
  • Connect to home WiFi network
  • Assign location/room in app
  • Complete: Device available in dashboard
🌡️

Pairing Health Monitors

Duration: 2-4 minutes

  • Initialize device with batteries/charging
  • Connect to home WiFi (if applicable)
  • VIS App → Health Devices → Pair
  • Select manufacturer and model
  • Follow device-specific verification
  • Complete: Readings sync automatically
