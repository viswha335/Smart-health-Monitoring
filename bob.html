❤️

Heart Rate Monitoring (HR Module)

Advanced real-time HR analysis engine with clinical-grade monitoring. Detects bradycardia (<60 bpm), tachycardia (>100 bpm), and critical conditions requiring immediate attention.

Capabilities:
• Normal Range: 60-100 bpm (Athletes: 40-60)
• Heart Rate Variability (HRV) calculation
• Anomaly detection & spike alerts
• Fitness level assessment
📊

Multi-Device Fusion

Intelligent aggregation from Apple Watch, Fitbit, Garmin, and other wearables. Conflict resolution using timestamp priority and sensor accuracy scoring.

Features:
• Device-adaptive thresholds
• Signal quality tracking (0-1)
• Activity-aware analysis
• Auto-reconnect on disconnection
⚠️

Alert & Recommendation Engine

Real-time health alerts with personalized recommendations. Critical alerts trigger emergency notifications for severe bradycardia/tachycardia.

Alert Levels:
• Critical: <40 or >140 bpm
• Warning: Out-of-range at rest
• Normal: Healthy range
• Exercise mode: Activity-based
🎯

Target Heart Rate Zones

Age-adjusted exercise zones using Karvonen formula. Calculates optimal training intensities for moderate, vigorous, and high-intensity workouts.

Training Zones:
• Moderate: 50-70% max HR
• Vigorous: 70-85% max HR
• High: 85-100% max HR
• Max HR = 220 - age
Layer 03

Data Processing Layer

Raw sensor signals are transformed through a four-stage pipeline — validated, cleaned, enriched with features, and normalized — before feeding into AI models.

01

Data Validation

Schema checks, range validation, and completeness verification ensure only well-formed sensor packets enter the pipeline.

Schema CheckRange Guard
02

Data Cleaning

Bandpass filtering removes motion artefacts. Median filters suppress PPG noise. Missing samples interpolated via cubic spline.

Noise RemovalArtefact FilterInterpolation
03

Feature Extraction

MFCC coefficients for speech emotion, Facial Action Units for fatigue, HRV spectral features, and step cadence metrics are extracted in parallel.

MFCCFAU MappingHRV Features

Normalization & Scaling

Z-score normalization aligns features across modalities. Min-max scaling constrains values for neural network inputs.

Layer 04

AI Intelligence Layer

Four specialized machine learning models run in parallel, each optimized for a distinct health-analysis domain.

⚡

All four models execute simultaneously in parallel threads for sub-100ms total inference time

🌳
Random Forest

Activity Classification

Classifies physical activities — walking, running, resting, cycling — from accelerometer and gyroscope features with 97.4% accuracy across 12 activity classes.

🫀
SVM

Cardiovascular Risk

RBF-kernel Support Vector Machine predicts short-term cardiac risk from HRV, SpO₂, and blood pressure trend features with clinical-grade precision.

🎙️
CNN — Speech

Emotion & Stress Detection

1D Convolutional Neural Network trained on MFCC audio features detects emotional state — stress, calm, anxiety, fatigue — from 2-second voice samples.

👁️
CNN — Image

Fatigue Detection

Vision-based CNN analyzes facial landmarks and eye aspect ratio from front-camera frames to detect micro-expressions of fatigue and drowsiness in real time.

Model Fusion

Ensemble Decision Fusion

Outputs from all four models are combined using a weighted ensemble strategy. Confidence scores from each classifier are fused with patient history priors to produce a unified health risk score.

Layer 05

Decision & Recommendation Engine

AI model outputs are evaluated against dynamic thresholds and patient baselines to generate clinically-relevant, actionable recommendations.

⚠️ Health Risk Alerts

🚨

Critical Medical Alert

Heart rate exceeds 180 BPM or SpO₂ drops below 90% — emergency notification dispatched to contacts and nearby services.

⚡

Elevated Cardiovascular Risk

SVM confidence >75% for high cardiac risk — recommends rest, hydration, and optional GP consultation within 24 hours.

😰

Stress & Emotion Warning

Sustained high-stress emotion detected over 30-minute window — triggers breathing exercises and mindfulness notification.

✅

All Vitals Normal

All metrics within baseline thresholds — daily health score calculated and displayed on dashboard with achievement badges.

💡 Personalized Recommendations

🏃‍♂️

Exercise Suggestions

😴

Rest Recommendations

🧘

Stress Management

Medical Consultation

Application Layer

Cross-Platform UI

React Native mobile app with responsive web dashboard. Consistent experience across iOS, Android, and browser without data loss.
